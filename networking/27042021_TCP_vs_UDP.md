#### UDP

```js
PROS:
  - Smaller packets
  - Less bandwidth
  - Stateless

CONS:
  - No Acknowledgment
  - No Guaranteed Delivery
  - Connectionless
  - No congestion control
  - No Ordered packets
  - Security
```

#### TCP
```js
  PROS:
  - Acknowledgment
  - Guranteed Delivery
  - Connection based
  - congestion control
  - Ordered packets

  CONS:
  - Larger packets
  - More bandwidth
  - slower then UDP
  - stateful
  - Server memory DDOS
```
